Per Page

Request URL pattern : https://quran.rf.gd/per-page.php?params=values

Request type: GET

Parameter Description Data Type Required Example
page Number 1 through 604. Default if not set is 604 numeric optional https://quran.rf.gd/per-page.php?page=1
font_size Default if not set is 30 numeric optional https://quran.rf.gd/per-page.php?font_size=30

Response format : json

Per Ayah

Request URL pattern : https://quran.rf.gd/per-ayah.php?params=values

Request parameters : surah, ayah, font_size.

Request values format : Numeric, surah (1-114)

Request type: GET

Response format : json

Example : https://quran.rf.gd/per-ayah.php?surah=1&ayah=1&font_size=30
